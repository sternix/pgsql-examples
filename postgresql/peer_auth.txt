peer authentication'da aynı hostta local olmayan bir kullanıcı ile bağlanabilmek için

\c $DBNAME $USERNAME 127.0.0.1
\connect $DBNAME $USERNAME 127.0.0.1

ile bağlanabiliriz

\c or \connect [ -reuse-previous=on|off ] [ dbname [ username ] [ host ] [ port ] | conninfo ]

=> \c mydb myuser host.dom 6432
=> \c service=foo
=> \c "host=localhost port=5432 dbname=mydb connect_timeout=10 sslmode=disable"
=> \c -reuse-previous=on sslmode=require    -- changes only sslmode
=> \c postgresql://tom@localhost/mydb?application_name=myapp
